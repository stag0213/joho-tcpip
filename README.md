# TCP/IP 4階層モデル - インターネットプロトコル完全ガイド

インターネットの4階層プロトコル（TCP/IPモデル）とデータの流れを視覚的に学べる教育用Webサイトです。

## 🌟 プロジェクト概要

このWebサイトは、TCP/IP 4階層モデルを初心者から中級者まで理解できるように、視覚的でインタラクティブな方法で解説します。ネットワークの基礎を学ぶ学生、エンジニア、技術者に最適な学習リソースです。

## ✨ 主な機能

### 1. **4階層の詳細解説**
- **アプリケーション層**: HTTP、HTTPS、FTP、SMTP、DNSなどのプロトコル
- **トランスポート層**: TCP（信頼性重視）とUDP（速度重視）の違い
- **インターネット層**: IP、ICMP、ARPによるルーティングとアドレス管理
- **ネットワークインターフェース層**: Ethernet、Wi-Fiなどの物理接続

### 2. **インタラクティブなデータフローアニメーション**
- **カプセル化プロセス**: 送信時にデータが各階層でヘッダーを追加される様子を可視化
- **デカプセル化プロセス**: 受信時にヘッダーが除去される様子を可視化
- リアルタイムでパケットの変化を確認できる動的アニメーション

### 3. **実用的な実例**
- **Webブラウジング**: HTTPSでWebページを表示する際のプロトコルスタック
- **メール送信**: SMTPによるメール配送の仕組み
- **オンラインゲーム**: UDPを使用する理由と利点

### 4. **ヘッダー情報の詳細説明**
- Ethernetヘッダー（MACアドレス）
- IPヘッダー（IPアドレス、TTL）
- TCPヘッダー（ポート番号、シーケンス番号）
- FCS（エラー検出）

### 5. **レスポンシブデザイン**
- デスクトップ、タブレット、スマートフォンに対応
- モバイルデバイスでも快適に閲覧可能

## 🚀 使い方

### ローカル環境での実行
1. プロジェクトファイルをダウンロード
2. `index.html` をWebブラウザで開く
3. ナビゲーションメニューから各セクションに移動
4. 「データの流れ」セクションでボタンをクリックしてアニメーションを体験
